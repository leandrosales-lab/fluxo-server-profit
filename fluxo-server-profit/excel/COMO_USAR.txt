COMO USAR A MACRO VBA — FLUXO BRIDGE
======================================

Ha duas versoes da bridge:
  - FLUXO_BRIDGE.bas     → usa DDE (legado)
  - FLUXO_BRIDGE_RTD.bas → usa RTD (recomendado)

O Profit Pro recomenda RTD para garantir funcionamento correto.
DDE e RTD NAO podem ser usados ao mesmo tempo.


======================================
OPCAO 1: RTD (RECOMENDADO)
======================================

1. ABRIR O EXCEL
   - Criar um arquivo novo ou usar FLUXO_BRIDGE.xlsm (salvar como macro habilitada)
   - Habilitar macros ao abrir

2. IMPORTAR O MODULO VBA
   - Alt+F11 → abre o Editor VBA
   - Clique direito em "Modulos" → Importar Arquivo
   - Selecionar: FLUXO_BRIDGE_RTD.bas
   - Fechar o Editor VBA

3. HABILITAR RTD NO PROFIT PRO
   - No Profit: menu Ferramentas > Exportar em Tempo Real
   - Selecionar modo RTD (nao DDE)
   - Adicionar o ativo desejado (ex: PETR4)

4. INSERIR FORMULAS RTD (automatico ou manual)

   AUTOMATICO:
   - Alt+F8 → selecionar "InserirFormulasRTD" → Executar
   - Digitar o ativo desejado (ex: PETR4)
   - As formulas serao inseridas em E1:I1

   MANUAL (inserir nas celulas do Excel):
   E1 = =RTD("rtdtrading.rtdserver";;"PETR4_F_0";"ULT")   (ultimo preco)
   F1 = =RTD("rtdtrading.rtdserver";;"PETR4_F_0";"VOL")   (volume)
   G1 = =RTD("rtdtrading.rtdserver";;"PETR4_F_0";"ACP")   (agente comprador)
   H1 = =RTD("rtdtrading.rtdserver";;"PETR4_F_0";"AVD")   (agente vendedor)
   I1 = =RTD("rtdtrading.rtdserver";;"PETR4_F_0";"NEG")   (num negocios)

   SUFIXOS:
     _F_0 = Bovespa (acoes: PETR4, VALE3, MGLU3, etc.)
     _B_0 = BMF (futuros: DOLFUT, WINFUT, INDFUT, etc.)

   Exemplo para mini-dolar:
   E1 = =RTD("rtdtrading.rtdserver";;"WDOFUT_B_0";"ULT")

   NOTA: Se usar Excel em portugues, o separador e ";" (ponto-virgula).
         Se usar Excel em ingles, use "," (virgula).

5. INICIAR O SERVIDOR PYTHON
   No PowerShell:
     cd C:\Users\usuario\Downloads\fluxo-server-profit
     python main.py --asset PETR4 --modo manual --port 5000

6. INICIAR A BRIDGE
   - No Excel: Alt+F8 → selecionar "IniciarBridge" → Executar
   - O VBA lera E1:I1 a cada 500ms e enviara ao servidor Python

7. LINKAR OS PARAMETROS DO ROBO NTSL
   No Profit, ao configurar o robo robo_fluxo_v2_com_servidor.ntsl,
   linkar via DDE as celulas do Excel:

   iSinalExterno   → celula A1 desta planilha
   fConfiancaSinal → celula B1 desta planilha
   fPrecoSinal     → celula C1 desta planilha
   fStopSinal      → celula D1 desta planilha

   Formula DDE no Profit para ler celula A1 da planilha "Plan1":
   =Excel|[FLUXO_BRIDGE.xlsm]Plan1!R1C1

8. SEQUENCIA DE INICIALIZACAO
   a) Iniciar servidor Python:
      python main.py --asset PETR4 --modo manual --port 5000

   b) Abrir Excel e rodar InserirFormulasRTD (se ainda nao fez)

   c) Rodar IniciarBridge()

   d) Abrir Profit Pro e iniciar o robo NTSL em simulacao

9. PARAR
   - Alt+F8 → "PararBridge" → Executar
   - O sinal e zerado (WAIT) automaticamente


======================================
OPCAO 2: DDE (LEGADO)
======================================

1. ABRIR O EXCEL
   - Criar um arquivo novo ou usar FLUXO_BRIDGE.xlsm (salvar como macro habilitada)
   - Habilitar macros ao abrir

2. IMPORTAR O MODULO VBA
   - Alt+F11 → abre o Editor VBA
   - Clique direito em "Modulos" → Importar Arquivo
   - Selecionar: FLUXO_BRIDGE.bas
   - Fechar o Editor VBA

3. INSERIR FORMULAS DDE DO PROFIT (celulas E1:I1)
   Abrir o Profit Pro e inserir nas celulas do Excel:

   E1 = =Profit|Ultima!PETR4       (ultimo preco)
   F1 = =Profit|VolNeg!PETR4       (volume do ultimo negocio)
   G1 = =Profit|CodAgente!PETR4    (codigo do agente/corretora)
   H1 = =Profit|TipNeg!PETR4       (tipo: C=compra, V=venda)
   I1 = =Profit|QtdNeg!PETR4       (quantidade de negocios)

   NOTA: Troque PETR4 pelo ativo desejado.
   NOTA: O Profit deve ter o servidor DDE habilitado:
         Ferramentas > Configuracoes > DDE > Habilitar servidor DDE

4. INICIAR A BRIDGE
   - No Excel: Alt+F8 → selecionar "IniciarBridge" → Executar
   - O VBA lera E1:I1 a cada 500ms e enviara ao servidor Python

5. LINKAR OS PARAMETROS DO ROBO NTSL
   (mesmo procedimento da opcao RTD, passo 7)

6. SEQUENCIA DE INICIALIZACAO
   a) Iniciar servidor Python:
      python main.py --asset PETR4 --modo dde

   b) Abrir Excel e iniciar macro IniciarBridge()

   c) Abrir Profit Pro e iniciar o robo NTSL em simulacao

7. PARAR
   - Alt+F8 → "PararBridge" → Executar
   - O sinal e zerado (WAIT) automaticamente
